Prescription Analyzer with Google Generative AI

This project allows users to upload prescription images and analyze them using Google’s Generative AI gemini-pro-vision model. The application converts uploaded images into base64 format and feeds them to the AI model to generate a text-based analysis of the prescription content.

Features

Image Upload: Users can upload prescription images in common formats (PNG, JPG, JPEG).

AI Analysis: Utilizes Google’s gemini-pro-vision generative model to extract textual information from images.

Simple Interface: Minimal HTML and JavaScript setup for quick integration.

Real-Time Output: Displays AI-generated text directly on the webpage.
